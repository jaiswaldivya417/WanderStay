# WanderStay

An e-commerce platform for booking hotel rooms and shelter services online.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User authentication (JWT)
- Hotel/room listing with search & filters
- Online payment integration
- Booking management system

## Setup Instructions
1. Clone the repo
2. Run `npm install` in both client and server folders
3. Add `.env` variables
4. Run `npm start`
